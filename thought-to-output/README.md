# Thought to Output

个人 Agent Skill：将语音转文字、碎片笔记和发散表达整理为可读文字稿。

## 核心原则

**每一次「整理」，必定产出一篇文字稿。** 文字稿是主产出，结构化卡片是可选附加。

- 短输入 → 短文字稿（几段连贯文字） + 可选卡片
- 中等输入 → 中篇文字稿（有结构的短文） + 可选卡片
- 长输入 → 长篇成稿（完整文章）

## 使用方式

1. 查看 `SKILL.md` 获取完整定义
2. 参考 `references/` 了解输出类别和质量标准
3. 参考 `assets/sample-outputs.md` 查看具体示例

## 设计原则

- 文字稿是必选产出，卡片是可选附加
- 去重不去细节：删除重复表述，但保留具体的人、事、物、场景
- 多主题用暗线串联，不强行压成单主线
- 保留用户语气和第一人称视角
- 宁可少附加，也不硬凑类别
